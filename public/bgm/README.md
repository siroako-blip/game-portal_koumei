# BGM 音源の置き場所

ここに音声ファイルを置くと、対応するゲームでBGMとして再生されます。

| ファイル名 | 使われる場所 |
| --- | --- |
| `lostcities.mp3` | ロストシティ（`/lostcities` ロビー & 対戦画面） |

## 注意

- ファイル名は上表のとおり固定（`app/lostcities/components/BgmPlayer.tsx` の `BGM_SRC` を参照）。
- ループ再生されるため、シームレスにループする曲が望ましい。
- 著作権フリー / ライセンス的に再配布可能な音源を使うこと（本プロジェクトは非公式ファンプロジェクト）。
- 音量は既定で 0.4。変更は `BgmPlayer.tsx` の `VOLUME` を編集。
- ファイル未配置でもエラーにはならず、単に無音になる。
