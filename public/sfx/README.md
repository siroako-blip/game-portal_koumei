# 効果音（SFX）の置き場所

ここに音声ファイルを置くと、ロストシティの対戦画面で操作音として再生されます。
ファイル名は固定（`app/lostcities/components/sfx.ts` の `SFX_SRC` を参照）。

| ファイル名 | 鳴るタイミング |
| --- | --- |
| `select.mp3` | 手札のカードを選択したとき |
| `play.mp3` | 道にカードを出したとき |
| `discard.mp3` | カードを捨てたとき |
| `draw.mp3` | 山札・捨て札から1枚引いたとき |

## 注意

- 短い単発音（数百ミリ秒）が望ましい。
- 著作権フリー / 再配布可能な音源を使うこと（本プロジェクトは非公式ファンプロジェクト）。
- 音量は既定で 0.5。変更は `sfx.ts` の `VOLUME` を編集。
- ファイル未配置でもエラーにはならず、単に無音になる。
- ON/OFF は対戦画面左下の 🔔/🔕 ボタンで切り替え（設定は localStorage に記憶）。
